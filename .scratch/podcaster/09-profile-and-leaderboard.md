# 09 — Profile + Top Podcasters leaderboard

**Triage:** ready-for-agent

## What to build

A creator profile and a public leaderboard of prolific creators.

- A profile page showing the creator's listener count and a grid of their podcasts, with an
  empty state before they publish anything.
- A newly published episode appears on the author's profile.
- A **Top Podcasters** leaderboard ranked by number of published podcasts (reactive join).

## Acceptance criteria

- [ ] Profile shows the creator's listener count and a grid of their podcasts
- [ ] Profile shows an empty state when the creator has published nothing
- [ ] A published episode appears on the author's profile
- [ ] Top Podcasters leaderboard ranks creators by podcast count and updates reactively
